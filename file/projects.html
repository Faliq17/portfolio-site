Projects

Dynamic Exrate - Power BI

This dashboard details Malaysia's monetary policy, covering Currency Circulation trends from 2000-2023 and listing Exchange Rates for the MYR against 159 currencies. It provides key forecasts for the MYR's trend against neighboring (IDR, SGD, THB) and major global currencies (USD, CNY) up to 2030.

Parcel Management System - Power Apps

This application is the EZTECH Parcel Management System, designed to streamline parcel services across three roles: Student, Employee, and Admin. Its core functions allow employees to enter parcel details and update payment status (Paid/Unpaid), while students can track their parcels and make payments, including options for Cash on Delivery.

Kiosk Prihatin UiTM - Figma

The KIOSK PRIHATIN UiTM prototype app is a student welfare tool that allows users to view the campus kiosk's real-time item stock (like food essentials) and latest information. Its main functions are enabling donations towards specific items and providing a Volunteer Form for students to help run the initiative.

Banana Tech Solution - Figma

This admin dashboard provides a complete management system for an e-commerce smartphone store. It features a Dashboard summarizing sales, stock, and weekly orders. Administrators can manage Products, track Orders, view Customer data, monitor Transactions, and analyze Reviews and ratings.

The Smart Homework Assistant - Figma

The Smart Homework Assistant app is an all-in-one student tool offering assignment management, a Math Solver (evaluate, differentiate, graph), and tutor assistance. It includes a Findings feature for research that generates APA style citations, and a Dashboard to track progress, tasks, and study time.
